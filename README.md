# Maze-rat-solver

## Description
This is a Python program using the `tkinter` library that generates a random maze and uses a recursive backtracking algorithm to help a "mouse" (a circular icon) find its way from the entrance to the exit.

## How to Run
1. Ensure you have Python installed on your computer.
2. Download the `maze_solver.py` file.
3. Open your terminal or command prompt.
4. Run the command: `python maze_solver.py`

## Features
* **Random Maze Generation:** Every time you run the program, a new maze is created.
* **Visual Solving:** You can watch the mouse move through the maze in real-time.
* **Backtracking:** If the mouse hits a dead end, it highlights the path and tries a different route.
